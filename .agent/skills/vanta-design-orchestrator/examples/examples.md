## Ejemplo 18: Ciclo de Validación + Análisis de Causa Raíz

**Skills involucradas:** 4

```
# DISCOVERY: 5 Whys + Brechas
strategy/validation-sustainability §2 (5 porqués + brechas de experiencia) →
strategy/business-model-design (revisar BMC con hallazgos)

# VALIDACIÓN: Ciclo EVP
strategy/validation-sustainability §1 (hipótesis → MVP → test → escalar/pivotar) →
strategy/lean-design (scope MVP mínimo) →
prototyping-testing (plan de validación) →
strategy/metrics-framework (medir impacto)
```

**Resultado:** Validación estructurada de hipótesis de diseño con causa raíz identificada, MVP scope definido, plan de test A/B estratégico, y decisión informada (escalar/pivotar/iterar).

**Outputs:**
- Cadena de 5 porqués con causa raíz documentada
- Hipótesis de diseño en formato "Creemos que [X] producirá [Y]"
- Plan de validación con criterio de éxito y n mínimo
- Registro de validación (V-2026-XXX)
- Decisión documentada: escalar, pivotar o iterar

**Tiempo estimado:** 2-4 horas

## Ejemplo 20: Investigación + PRD + Prototipado (skills.sh Workflow)

**Skills involucradas:** 5 (3 skills.sh + 2 Vanta)

```
# FASE 0: Extraer tokens de competencia + research
extract-design-system (npx extract-design-system <url_competencia>) →
just-scrape (scrape contenido + competitive intelligence)

# FASE 1: Sintetizar hallazgos en PRD
to-prd (conversación → PRD estructurado + issues)

# FASE 2: Validar con prototipo desechable
prototype (UI variations para explorar direcciones) →
strategy/validation-sustainability §1 (validar hipótesis)

# HANDOFF a producción
handoff (decisión log + hallazgos + next steps) → frontend-design
```

**Resultado:** Pipeline completo de competitive intelligence → PRD → prototipado → handoff a producción.

**Outputs:**
- `tokens.json` + `tokens.css` extraídos de sitio de referencia
- Competitive intelligence report (scraped + structured)
- PRD en issue tracker con prioridades alineadas a BMC
- Prototipo desechable validado (o descartado)
- Documento de handoff para siguiente agente/fase

**Tiempo estimado:** 2-4 horas
